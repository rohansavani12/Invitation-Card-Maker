# My-Android-Projects
Invitation Card Maker is a simple and customizable Android application that allows users to create digital invitation cards effortlessly. Users can choose templates, add text, stickers, and images, and export high-quality invitation cards for events like birthdays, weddings, and parties. Built with Java/Kotlin, XML, and Firebase (if applicable), this project demonstrates UI/UX design, image processing, and event-based customization in Android development.
